- Implemented previously missing recursive-glob functionality in the file
monitor. It is now possible to use double asterisk (`**`) in a glob to indicate
recursive directory traversal (much like in most Unix-style shells). Just like
elsewhere in Cabal (and unlike most shells), `**` is limited to the last
directory component of a path, so `/foo/bar/**/*.txt` is valid, but
`/foo/**/bar/*.txt` is not.
