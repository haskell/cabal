synopsis: Add PackageMeta_ autogenerated module
packages: Cabal
prs: #11756

description: {
Add a new autogenerated module `PackageMeta_<pkgname>` that exposes
build-environment metadata: compiler identity, platform (OS and
architecture), cabal flag values, and (stubbed) VCS revision info.
This complements `PackageInfo_` (static `.cabal` metadata) and
`Paths_` (install paths).
}
