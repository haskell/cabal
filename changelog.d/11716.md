---
synopsis: Adding explicit optimization flags for interaction with GHC
packages: [Cabal, cabal-install]
prs: 11716
---

Cabal now adds explicit `-optc-O2`, `-optcxx-O2` and `-opta-O2` flags when
invoking GHC for C, C++ and assembler source files that are built with
optimisation.  These explicit flags are only inserted when the user has not
already provided an optimisation flag (`-O`, `-O0`, `-O1`, `-O2`, `-O3`),
so that existing user choices are never overridden or duplicated.
