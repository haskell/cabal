---
synopsis: Drop support of building with GHC < 9.4
packages: [Cabal-syntax, Cabal, cabal-install-solver, cabal-install]
prs: 11966
---

The support window of GHCs, which can be used to build Cabal, has been narrowed from 5 to 3 years.
We now support only GHC 9.4+.

`cabal-install` remains capable to drive older GHCs, as long as supporting it takes minimal effort.
